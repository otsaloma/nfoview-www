Release details and other downloads

NFO Viewer is a simple viewer for NFO files, which are “ASCII” art in the CP437 codepage. The advantages of using NFO Viewer instead of a text editor are preset font and encoding settings, automatic window size and clickable hyperlinks.

NFO Viewer is available for Linux and Windows, released under the GNU General Public License (GPL). The user interface is based on the GTK+3 toolkit and has been designed to best fit the GNOME desktop environment.
